### Current Capabilities
- Automated meal notifications (45 min before each meal)
- Fetch lyrics from Genius API  
- Display mess menu with slash commands
- Connect to voice channels
- Chat through CLI

### Menu Commands (Slash Commands)
```
/menu - Get full week's mess menu
/today - Get today's mess menu
/breakfast - Show today's breakfast menu
/lunch - Show today's lunch menu  
/snacks - Show today's snacks menu
/dinner - Show today's dinner menu
```

### Other Commands
```
lyr song name artist - Fetch lyrics for a song
/ping - Test bot response
/help - Show this help menu
```

### CLI Commands (For Admins)
```
node luci.js - Start the main bot
node vc.js - Connect to voice channel
```

### Auto Notifications
Bot automatically sends meal reminders 45 minutes before:
- Breakfast: 8:00 AM - 9:00 AM
- Lunch: 12:30 PM - 2:30 PM  
- Snacks: 5:00 PM - 6:00 PM
- Dinner: 8:00 PM - 9:00 PM
